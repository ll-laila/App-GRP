# ERP-GS

## Getting Started

### Installation

1. Clone the repository or download the ZIP file.
```bash
git clone [repository-url]
```

2. Install necessary dependencies.
```bash
mvn install
```

## Usage

```bash
./mvnw spring-boot:run
```
